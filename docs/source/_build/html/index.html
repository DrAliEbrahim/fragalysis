Fragalysis

Fragalysis is a powerful app that can be used to visualise, analyse, and curate experimental and virtual fragment-screening data from inside your browser.

Getting started

Crystallographic data are uploaded to Fragalysis via XChem Align using either the production or staging stacks. If you are new to Fragalysis, we recommend starting with the Fragalysis user guide to familiarise yourself with the most common analysis tasks.
